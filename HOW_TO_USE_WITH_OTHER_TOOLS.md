# How to Use These Notes With Other Tools

Your AI memory is stored as simple `.md` files so it isn’t locked into one app.

## Use with AI tools

You can:

- open a note  
- copy its contents  
- paste it into your AI chat as context before you ask a question  

For bigger tasks, you can combine several notes into one “Copy for AI” block (for example: company context + writing style + current project).

## Move notes into other apps

### Obsidian

Obsidian works directly with Markdown files.[web:136][web:137]

- Create or open a vault in Obsidian.  
- Drop your `.md` files or folders into the vault folder on disk.  
- They appear as notes inside Obsidian.

### Notion

Notion can import Markdown files and folders.[web:150]

- Export selected notes from this vault (or download a folder/ZIP).  
- In Notion, use **Import** and choose your files or ZIP.  
- Notion turns each `.md` file into a page.

### ClickUp

ClickUp Docs supports importing Markdown.[web:148][web:142]

- Export or download the `.md` file you want.  
- In ClickUp Docs, choose **Import** and select the Markdown option.  
- Your note becomes a doc.

## Simple rule

Write notes once in this vault.  
Reuse them with any AI or note tool you like.

If your tools change, your Markdown files can come with you.

# Reusable Prompts

A collection of prompts I use regularly. Copy the block and paste into any AI assistant.

---

## Code Review

```
Review this code for:
1. Bugs or edge cases I might have missed
2. Performance issues
3. Readability improvements
4. TypeScript best practices

Be specific. Point to line numbers where possible. Don't rewrite the whole thing unless asked.

[PASTE CODE HERE]
```

---

## Feature Planning

```
I want to build: [FEATURE DESCRIPTION]

My stack: Next.js, TypeScript, Tailwind, Supabase

Give me:
1. A simple implementation plan (max 5 steps)
2. Any gotchas or edge cases to watch out for
3. Suggested file structure

Keep it practical. I want to ship this in a day.
```

---

## Debugging Help

```
I'm getting this error: [ERROR MESSAGE]

Context:
- What I was doing: [DESCRIPTION]
- Stack: [STACK]
- What I've already tried: [ATTEMPTS]

Here's the relevant code: [CODE]

What's causing it and how do I fix it?
```

## Writing a README

```
Write a README for this project:

Project name: [NAME]
What it does: [DESCRIPTION]
Tech stack: [STACK]
Target user: [USER]

Include: overview, quick start, features list. Keep it under 200 words. No fluff.
```

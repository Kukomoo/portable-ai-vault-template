# Why Markdown

## What is a `.md` file?

A Markdown file is a simple text document that ends in `.md`.

You can think of it like a lighter version of a Word or Google doc:

- It still has titles, bullet points, and sections.
- It uses plain text with a few symbols (like `#` for headings and `-` for lists).[web:109][web:112][web:115]
- It stays readable even without any special app.

## Why use it for AI memory?

Markdown works well here because it is:

- easy to read and edit as plain text  
- easy to copy into AI chats  
- easy to organize into folders  
- easy for many tools to work with and import later[web:104][web:105][web:110]

## A tiny example

```md
# Writing style

## Preferences
- Keep answers concise.
- Use bullet points instead of long paragraphs.
- Give startup-focused examples.
- End with practical next steps.
```

This is enough for an AI to understand how you like to read answers.

## Where else can you use `.md` files?

Many tools already handle Markdown well:

- GitHub renders Markdown files directly in the browser.[web:104][web:107]  
- Obsidian is built around Markdown notes.[web:136][web:137]  
- Notion can import text and Markdown files.[web:150]  
- ClickUp Docs can import Markdown into docs.[web:148]

That means your AI memory can move with you if your tools change.
